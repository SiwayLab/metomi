# Metomi Frontend

Metomi 是一款极简而强大的个人图书管理系统。前端采用 **Vue 3** (Composition API) 和 **Vite** 构建，结合 **Element Plus** 提供现代化的用户界面体验。

## 特性
- 支持本地 PDF / EPUB 电子书导入与阅读。
- 基于 ISBN 的豆瓣元数据抓取。
- 支持图书状态、海市蜃楼毛玻璃效果、深色模式支持（开发中）。
- 高性能的多文件管理、瀑布流图书浏览。
- 分层配置与设置存储。

## 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 生产环境构建
npm run build
```

## 技术栈
- Vue 3 (`<script setup>`)
- Vite
- Element Plus
- Pinia (状态管理)
- Vue Router (前端路由)
- Axios
